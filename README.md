## Setup Instructions

### Prerequisites

- Java 11 or higher
- Maven 3.6 or higher
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/gradetracker.git
   cd gradetracker
   ```

2. Build the project:
   ```bash
   mvn clean package
   ```

3. Run the application:
   ```bash
   mvn javafx:run
   ```
